# Wear Mixue

[English](README.md)

用手表点蜜雪冰城。

Wear Mixue 使用蜜雪冰城相关接口实现手表端点餐，可以在手表上浏览门店商品、选择规格、提交已购团购券订单，并正常显示取餐码。适合在校高中生这类不方便频繁拿出手机、但又想提前点好饮品的人群：提前在快手极速版买好蜜雪冰城团购券，到店前用手表下单，出示取餐码取餐即可。

## 功能

- 在手表端浏览蜜雪冰城商品列表。
- 支持选择商品规格，例如温度、糖度等。
- 使用快手极速版蜜雪冰城小程序的授权登录信息登录。
- 支持提前购买团购券后，在手表端完成下单。
- 下单后可以正常显示取餐码。
- 提供手机端辅助输入，方便扫码把登录 JSON 发送到手表。

## 使用前准备

使用前需要先在快手极速版里购买好蜜雪冰城团购券。当前流程依赖快手极速版蜜雪冰城小程序的授权信息，首次登录需要通过抓包获取一次登录 JSON。

你需要准备：

- 一台安装了本项目手表端的手表。
- 可选：一台安装了本项目手机端的手机，用于扫码发送登录 JSON。
- 快手极速版。
- 抓包软件，例如 Reqable。

## 首次登录

1. 在手机上打开抓包软件，例如 Reqable，并开始抓包。
2. 打开快手极速版，进入蜜雪冰城小程序。
3. 在小程序里授权手机号。
4. 回到抓包软件，搜索 `decrypt`。
5. 找到这个请求：

```text
POST https://third-activity.mxbc.net/activity/v1/app/ks/tiny-app/decryptPhoneNum
```

6. 点进请求详情，查看响应内容。
7. 复制完整响应 JSON。
8. 在手表端粘贴 JSON，或打开手机端扫码发送到手表。
9. 点击“解析 JSON”。
10. 点击“登录”。

登录成功后，就可以在手表上选择门店、浏览商品、选择规格并使用已经购买的团购券下单。
